# CodeWhisper (VS Code)

Select code (or place the cursor in a symbol) and get a **structured explanation** (summary, complexity, suggestions) plus optional **spoken output** via **ElevenLabs**.

## Setup

1. Open this folder in VS Code and run **Terminal → Run Task** or:

   ```bash
   cd codewhisper && npm install && npm run compile
   ```

2. **Run Extension Development Host**: press `F5` from `codewhisper` (or open Run and Debug and launch “Extension”).

3. In the Extension Development Host, run commands:

   - **CodeWhisper: Set LLM API Key** — OpenAI-compatible key (default base URL `https://api.openai.com/v1`).
   - **CodeWhisper: Set ElevenLabs API Key** — `xi-api-key` from ElevenLabs.
   - Set **CodeWhisper › ElevenLabs: Voice Id** in Settings (from the ElevenLabs voice library).

4. Optional: set **CodeWhisper › Audio Fallback** to `local` on macOS to use the `say` command if ElevenLabs is not configured.

## Usage

- **CodeWhisper: Explain Selection** (editor context menu, Command Palette, or `Ctrl+Alt+G` / `Cmd+Alt+G` with focus in the editor).
- If nothing is selected, the extension explains the **enclosing symbol**, or a **window of lines** around the cursor.
- **CodeWhisper: Stop Speaking** stops playback in the explanation panel.

## Privacy

Sending an explanation transmits the **selected (or derived) code** to your **LLM** provider. Spoken audio is generated by **ElevenLabs** from the model text. Do not use with secrets you are not allowed to send externally.
